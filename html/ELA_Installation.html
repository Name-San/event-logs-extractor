About

Extract Logs Automation (ELA) is a compiled PowerShell script designed for easy extraction of Windows logs. The main purpose of the script is to address the issue of being unable to retrieve logs due to user unavailability.

Installation Process

Please read the steps below to avoid unnecessary runtime error.

  1. Ensure to use C:\App as the destination folder.
  2. Add the corresponding device ID when prompted. (Refer to the database for the correct ID.)
  3. To test the automation functionality, execute the Get Logs task in Windows Task Scheduler.
  4. The installer may be flagged as a false positive by most antivirus software. Simply click "allow" to proceed with the installation.
  5. You can examine and edit the script in the specified destination folder.
